INTRODUCCION HTML

Este es un curso que pretende intruducir el lector en el mundo del
HTML, No se requieren conocimientos previos, se empezará desde cero.
NO hay que saber nada de programación para utilizar HTML, basta
aprender un par de cosas sencillas para empezar a crear páginas.

Es MUY FACIL aprender HTML, Tan sólo necesitas un editor de texto, un
curso de html y un navegador web (como netscape o interned explore),
ni siquiera necesitas estar conectado a Interned
